# Personal-Project---MacroPad
My first ever personal project - a simple MacroPad based off HackClub Blueprint's MAcroPad Tutorial 

The Macropad will include 5 keys and a rotary encoder and maybe a OLED display

Hopefully it works!

BOM for parts: 
- 5x white blank DSA keycaps
- 1x EC11 Rotary encoder
- 1 unsoldered Seeed XIAO RP2040
- 1x 0.91 inch OLED display
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
