# Personal-Project---MacroPad
My first ever personal project - a simple MacroPad based off HackClub Blueprint's MAcroPad Tutorial. I started this project because I wanted a sort of easy-ish first project to give me some more experience in STEM. Next year I am going to college so I think that this will help me learn the fundamentals and give me more experience on creating things. I also plan on actually using this mainly for using shortcuts on my computer. I want to make it so there is a ctrl+c and ctrl+v shortcut on two of the keys for sure. - 5x white blank DSA keycaps
- 1x EC11 Rotary encoder

- 1x 0.91 inch OLED display
